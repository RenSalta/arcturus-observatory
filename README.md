# Arcturus

Observatorio virtual interactivo para explorar estrellas, constelaciones y exoplanetas.
## Demo

https://rensalta.github.io/arcturus-observatory/
## Características

- Bóveda celeste interactiva
- Catálogo estelar basado en datos astronómicos reales
- Fichas informativas de estrellas
- Visualización de constelaciones
- Sistema de búsqueda de estrellas
- Landing page inmersiva

## Tecnologías

- HTML
- CSS
- JavaScript
- Canvas API
